图像预处理

1.图像平滑处理

平滑处理技术（也称为模糊处理）：均值滤波、高斯滤波、中值滤波和双边滤波。

1.1 均值滤波

均值滤波是指任意一点的像素值，都是周围N*N个像素值的均值。

均值滤波可以有效地去除噪声，但可能会导致图像变得模糊。

如下图：

1.2 中值滤波

在使用邻域平均法去噪的同时也使得边界变得模糊。而中值滤波是非线性的图像处理方法，在去噪的同时可以兼顾到边界信息的保留。选一个含有奇数点的窗口，将这个窗口在图像上扫描，把窗口中所含的像素点按灰度级的升或降序排列，取位于中间的灰度值来代替该点的灰度值。

中值滤波在去除椒盐噪声（即图像中随机出现的黑白点）时非常有效。

计算过程如下图所示：

1.3 高斯滤波

为了克服简单局部平均法的弊端(图像模糊)，目前已提出许多保持边缘、细节的局部平滑算法。它们的出发点都集中在如何选择邻域的大小、形状和方向、参数加平均及邻域各店的权重系数等。图像高斯平滑也是邻域平均的思想对图像进行平滑的一种方法，在图像高斯平滑中，对图像进行平均时，不同位置的像素被赋予了不同的权重。高斯平滑与简单平滑不同，它在对邻域内像素进行平均时，给予不同位置的像素不同的权值。

高斯滤波在去除噪声的同时，能够更好地保留图像的边缘信息。

###1.4 双边滤波

双边滤波是一种非线性的平滑处理方法，它结合了空间邻近度和像素值相似度。与高斯滤波不同，双边滤波在平滑图像的同时，能够保留图像的边缘信息。这是因为双边滤波不仅考虑像素之间的空间距离，还考虑像素值之间的差异。

同时引入距离和像素值权重差异。

2.图像形态学操作

2.1 膨胀

膨胀是将目标区域接触的背景点合并到该目标物中，使目标边界向外扩张的处理。膨胀可以用来填补目标区域存在的某些空洞，也可以用来消除包含在目标区域中的小颗粒噪声。膨胀处理是腐蚀处理的对偶。

步骤：

1.将预先设定好形状以及原点位置的结构元素移动到图像中可包容（可包容的意思是说：结构元素的覆盖范围必须完全落在图像内）的所有像素点
2.判断该结构元素覆盖范围内的像素值是否存在至少一个为1的点，如果存在，则膨胀后图像中与结构元素原点相同位置的像素值为1，如果该覆盖范围内所有的像素值为0，则膨胀后图像相同位置上的像素值为0
3.对原图中的所有像素重复进行步骤2

3.边缘检测

3.1 sobel算子、scharr算子和Laplacian算子

3.1.1 sobel算子

Sobel 算子使用两个 3x3 的卷积核分别计算图像在水平和垂直方向上的梯度：
$$
Gx = \left[
\matrix{
\ -1 &0 &1\
\ -2 &0 &2\
\ -1 &0 &1
}
\right]
$$

$$
Gy = \left[
\matrix{
\ -1 &-2 &-1\
\ 0 &0 &0\
\ 1 &2 &1
}
\right]
$$

$$
G = \sqrt{G_x+G_y}
$$

优点：计算简单，适合检测水平和垂直边缘。

缺点：对噪声敏感，边缘检测效果一般。

3.1.2 scharr算子

$$
Gx = \left[
\matrix{
\ -3 &0 &3\
\ -10 &0 &10\
\ -3 &0 &3
}
\right]
$$

$$
Gy = \left[
\matrix{
\ -3 &-10 &-3\
\ 0 &0 &0\
\ 3 &10 &3
}
\right]
$$

优点：对边缘的响应更强，适合检测细微边缘。

缺点：对噪声敏感。

3.1.3 Laplacian算子

3.2 Canny算法

步骤如下：

1、首先用高斯滤波平滑图像，以降低噪声对梯度计算的影响。

2、使用 Sobel 算子计算图像的梯度幅值和方向。

3、非极大值抑制（Non-Maximum Suppression, NMS）：沿着梯度方向，保留局部梯度最大的像素点，抑制其他像素点。

4、双阈值检测（Double Threshold）：使用两个阈值（低阈值和高阈值）来确定真正的边缘。高于高阈值的像素点被认为是强边缘，低于低阈值的像素点被抑制，介于两者之间的像素点如果与强边缘相连则保留。

5、边缘连接（Hysteresis）：通过滞后阈值处理，将弱边缘与强边缘连接起来，形成完整的边缘。
